# Mathematical Background

This page collects the compact formulas and source-level notes that explain the main special-function families exposed by OpenSpecFun.jl.

## Airy family

AMOS `ZAIRY` computes the Airy function `Ai(z)` and its derivative `Ai'(z)`.
For `|z| > 1`, the original Fortran routine evaluates these functions through modified Bessel functions:

```math
Ai(z) = \frac{1}{\pi\sqrt{3}}\sqrt{z}\,K_{1/3}\left(\frac{2}{3}z^{3/2}\right),
\qquad
Ai'(z) = -\frac{1}{\pi\sqrt{3}} z\,K_{2/3}\left(\frac{2}{3}z^{3/2}\right).
```

The translated implementation keeps the original split between the small-`|z|` power series and the large-`|z|` Bessel path described in the prologue of [amos/zairy.f](https://github.com/qchen-fdii-cardc/openspecfun/blob/main/amos/zairy.f).

The continuation helper [amos/zacai.f](https://github.com/qchen-fdii-cardc/openspecfun/blob/main/amos/zacai.f) states the analytic continuation used to move the `K`-Bessel function between half-planes:

```math
K(\nu, z e^{m\pi i}) = K(\nu, z)e^{-m\pi i\nu} - m\pi i\,I(\nu, z).
```

That relation is the reason `zacai` exists as a trimmed continuation path for the Airy implementation.

## Dawson family

The real Dawson integral used by the Faddeeva wrappers is

```math
F(x) = e^{-x^2}\int_0^x e^{t^2}\,dt.
```

The package implements the real-valued form directly in Julia and then lifts it into the complex-valued Faddeeva helpers. The relation used in the code is

```math
F(x) = \frac{\sqrt{\pi}}{2} e^{-x^2} \operatorname{erfi}(x).
```

The more general complex-valued routines are mapped back to [Faddeeva/Faddeeva.c](https://github.com/qchen-fdii-cardc/openspecfun/blob/main/Faddeeva/Faddeeva.c) through the provenance table in the module.
